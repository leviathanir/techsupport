# TechSupport - Чат-бот поддержки

Виртуальный помощник интернет-магазина техники. Бот отвечает на вопросы о товарах, гарантии, возврате, доставке и оплате.

## Функциональность

- ✅ Отправка сообщений по кнопке и клавише Enter
- ✅ Индикатор "бот печатает" с задержкой ответа
- ✅ Блокировка отправки во время "печатания" бота
- ✅ Очистка чата
- ✅ Сохранение истории в localStorage
- ✅ Валидация формы (не пустое поле, максимум 500 символов)
- ✅ Обработка ошибок API
- ✅ Адаптивный дизайн
- ✅ Модульная структура JavaScript

## Технологии

- HTML5
- CSS3 (переменные, псевдоклассы, псевдоэлементы, анимации)
- JavaScript (ES6+, модули, async/await, Fetch API)
- LocalStorage для сохранения истории

## Установка и запуск

1. Скачайте все файлы проекта
2. Откройте `index.html` в современном браузере
3. Начните общение с ботом!

## Особенности реализации

### CSS
- Использованы CSS переменные для цветов, шрифтов и размеров
- Применены псевдоклассы (:hover, :focus, :disabled, :valid, :invalid)
- Использованы псевдоэлементы (::before, ::after) для декоративных элементов
- Адаптивная верстка с медиа-запросами
- Анимации для индикатора печатания и появления сообщений

### JavaScript
- Модульная структура с разделением на логические блоки
- Асинхронные операции с async/await
- Имитация API запросов с задержкой
- Обработка ошибок с try/catch
- Валидация пользовательского ввода
- Сохранение и загрузка истории из localStorage

## Команды бота

Бот понимает следующие ключевые слова:
- привет, здравствуйте
- гарантия
- возврат
- цена
- доставка
- оплата
- контакты
- спасибо
- пока
